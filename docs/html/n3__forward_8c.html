double * __n3l_forward_layer ( N3LLayer *  layer,
double *  inputs 
)

Internal function to execute forward propagation from the current layer to the next one.

This function first execute all neurons in the layer using concurrents threads. When all threads are executed, get the outputs for each neuron in the next layers.

Parameters
layerCurrent layer to execute.
inputsCurrent layer inputs.
Returns
Current layer outputs.
