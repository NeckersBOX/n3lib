void * __n3l_backward_execute ( void *  arg)

Internal function to execute backward propagation from the current layer to the previous one.

Recursive function to execute backpropagation from the current layer to the previous one, only if the layer passed as thread data is not NULL. After backpropagate it adjusts the current layer's weights.

Parameters
argPointer to an initialized __n3l_backward_data struct
Returns
No value returned.

◆ n3l_backward_propagation()

bool n3l_backward_propagation ( N3LNetwork *  net)

Execute backward propagation on the whole network.

Each call to the previous layer from the last layer is executed with concurrents threads.

Note
The member net->targets must be initialized before calling this function.
This function should be called after n3l_forward_execute()
Parameters
netInitialized network
Returns
TRUE if was correctely executed, otherwise FALSE.
